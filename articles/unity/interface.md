# Обзор Интерфейса Unity

## Окно Hierarchy (Иерархия)
Это список всех объектов, которые находятся в вашей сцене прямо сейчас.

**Что здесь делать:**
* Создавать новые объекты (свет, кубы, персонажей).
* Выбирать объект, чтобы увидеть его свойства.
* Группировать объекты (перетаскивая один внутрь другого).
* Удалять объекты из сцены.
