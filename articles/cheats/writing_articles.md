# 📝 Полный гайд: Создание контента для VR Kvant

Этот документ поможет вам быстро освоить формат оформления уроков и шпаргалок для нашего портала. Мы используем **Markdown** для текста и **JSON** для управления структурой сайта.

---

## 1. Основы оформления (Markdown)

Markdown — это текстовый стандарт, который превращается в красивую верстку на сайте.

### Заголовки
* `# Название статьи` — Используется только ОДИН раз в самом начале файла.
* `## Заголовок раздела` — Для крупных блоков.
* `### Подзаголовок` — Для уточнения внутри разделов.

### Списки
1. Используйте `1.`, `2.`, `3.` для нумерованных инструкций.
2. Порядок цифр будет соблюдаться автоматически.
* А если использовать `*` или `-`, то получится маркированный список.

### Акценты
* `**Жирный текст**` превратится в **Жирный текст** — используйте для терминов.
* `*Курсив*` превратится в *Курсив* — для примечаний.
* `> Цитата` — используется для полезных советов.

> **Важно:** Всегда оставляйте пустую строку до и после списков, цитат и блоков кода, чтобы они корректно отображались.

## 2. Вставка медиафайлов

### Изображения
Для вставки изображения используйте конструкцию:
`![Описание](img/папка_трека/название_файла.png)`

**Правила для картинок:**
1. Загружайте файлы в `img/blender/`, `img/unity/` или `img/unreal/`.
2. Названия файлов — только латиница (например, `screenshot_01.png`).

### Код (Скрипты)
Для вставки программного кода (например, C# для Unity) используйте три обратных апострофа и название языка. 
Пример:

```csharp
public class PlayerController : MonoBehaviour {
    void Update() {
        // Логика движения здесь
        Debug.Log("Hello VR!");
    }
}
```

### Видео (YouTube)
Наш сайт поддерживает вставку через `iframe`. На YouTube нажмите «Поделиться» -> «Встроить» и скопируйте код. Вставьте его прямо в файл.

## 4. Как опубликовать материал на сайте

Создание файла — это 50% работы. Чтобы сайт "увидел" статью, её нужно прописать в манифесте.

### Добавление Урока (в раздел Треки)
1. Откройте `articles/tracks.json`.
2. Найдите массив `lessons` внутри нужного трека.
3. Добавьте объект (не забывайте запятую перед ним, если он не первый):

```json
{ "title": "Название вашего урока", "file": "название_файла.md" }
```

### Добавление Шпаргалки
1. Откройте `articles/cheats.json`.
2. Добавьте новую запись в конец списка:

```json
{ "title": "Название шпаргалки", "file": "название_файла.md" }
```

## 💡 Чек-лист проверки (Перед "Commit")

* **Названия**: Файлы и папки названы латиницей (без пробелов, через `_`).
* **Пути**: Вы проверили пути до картинок.
* **Закрытие тегов**: Все блоки кода и жирный текст закрыты.
* **JSON**: Манифест-файлы сохранены корректно, нет лишних запятых в конце списков (перед `]`).
