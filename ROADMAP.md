# Архитектурный архив
### 1. Отказ от Tailwind CDN
**Статус:** Отложено. Согласно правилам проекта (`GEMINI.md`), использование npm-пакетов для Tailwind запрещено. CDN остается как наиболее легкий способ стилизации без этапа компиляции на стороне пользователя.

### 2. Оптимизация парсинга Markdown
**Статус:** Частично выполнено. Тексты статей теперь очищаются и кэшируются в `search_index.json` на этапе сборки, что ускоряет поиск. Полный пре-рендер HTML отложен для сохранения возможности динамической генерации ID виджетов.
